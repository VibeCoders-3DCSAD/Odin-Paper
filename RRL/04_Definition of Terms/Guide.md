# Format
* `[A/I/L]--[Author]_[and_Author2]_[et-al]_[Year]_[(NOT USED)].pdf`
##### ex. `L--Tambuli_and_Villarba_2026-(NOT USED).pdf`

# Labels
### Algo/Intl/Local
* A - Algorithm-specific
* I - International
* L - Local

### Author Names
* `[Author Surname]` if solo
* `[Author Surname 1]_and_[Author Surname 2]` if two authors
* `[Author Surname 1]_et-al` if three or more authors

### Year Published
* `[Year]` only, no month

### Used/Not Used
* No label if used within current version of paper
* `[Not Used]` if on standby
* `summarized` if summary of paper
