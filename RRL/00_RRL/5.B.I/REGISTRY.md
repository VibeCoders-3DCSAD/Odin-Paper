## I--Caroprese_et-al_2025

### Description

Introduces a stream-based data generator that generates user preferences and accommodates progressive changes in these preferences, modeling concept drift in dynamic data streams for recommender systems.
- Adapting to Shifts: Investigates how the fundamental model adapts to shifts in user behavior over time by altering the structure of the tripartite graph and modifying the underlying embeddings.
- Simulating Realistic Data: Develops a generalized method that simulates realistic data and generates streams with similar, yet controllable, drift dynamics, characterizing changes in user preferences over time.

## I--Gouni_2025

### Description

Explains that concept drift poses a significant challenge in financial fraud detection because legitimate transaction patterns evolve over time due to changing consumer behaviors and seasonal variations.
- Drift Adaptation Mechanisms: Addresses the need for adaptive solutions by employing ensemble-based approaches that combine multiple detection models with drift adaptation mechanisms, monitoring statistical properties, and implementing incremental learning techniques.
- Dynamic Behavioral Profiling: Details the application of dynamic behavioral profiling at both the individual and population levels to continuously update baseline patterns, distinguishing between legitimate behavioral evolution and fraudulent anomalies.

## I--Hovakimyan_&_Bravo_2024

### Description

Explores a wide range of methodologies, including drift detection methods (DDMs), window-based methods (WBMs), unsupervised statistical methods (USSMs), ensemble methods (EMs), and neural network (NN) techniques, which serve as the technical grounding for systems that update user profiles based on behavioral signal accumulation.
- Highlights Adaptive Learning: Discusses Adaptive Windowing (ADWIN) as a substantial contribution to adaptive learning strategies, relevant for progressively updating a user's financial behavioral profile as spending and income patterns evolve.
- Presents Neural Network Techniques: Examines the theoretical foundations of Neural Networks (NNs) in drift detection, emphasizing their adaptability and generalization capabilities for handling dynamic and evolving data environments.

## I--Liao_2026

### Description

Proposes a unified framework that tightly integrates concept drift detection mechanisms with incremental learning strategies, enabling models to identify distributional changes and update their knowledge without retraining.
- Dynamic Detection Approach: Investigates dynamic intrusion detection from the perspective of learning evolution to address limitations of traditional systems relying on static learning paradigms in non-stationary network environments.
- Adaptability Focus: Emphasizes continuity, stability, and adaptability in the framework, aiming to balance detection accuracy with computational efficiency under continuously changing conditions.

## I--Polytarchos_2025

### Description

Explains that a real-time data stream analysis system (BEReTiC) dynamically clusters transactions and customer profiles using a semi-supervised approach, detecting behavioral deviations and flagging anomalies through concept drift and similarity analysis.
- Detecting Behavioral Changes: Describes how the system detects concept drift when current data clusters differ significantly from the clusters identified in the previous round, enabling the identification of new behavioral patterns.
- Adaptive Profile Updating: Indicates that the system continuously updates clusters and 'fingerprints' (profiles) in real-time as new data streams in, demonstrating an adaptive learning strategy for handling concept drift in fraud patterns.

## I--Wu_et-al_2026

### Description

Studies a test-time adaptation (TTA) framework for causal time-series forecasting and direction classification applied to non-stationary data where the distribution changes over time.
- Financial Market Application: Applies the framework to daily equity and FX series (SPY, QQQ, EUR/USD) across different financial regimes, such as pandemic, high-inflation, and recovery, which serves as an analog to progressively updating a user's financial profile as patterns evolve.
- Adaptive Learning Strategies: Employs a quadratic drift penalty and an uncertainty-triggered fallback mechanism to maintain stability when updating only normalization affine parameters using recent unlabeled windows, serving as technical grounding for adaptive learning.

## I--Xiang_et-al_2023

### Description

Summarizes various concept drift adaptation methods under the deep learning framework, which are relevant to creating systems that update user profiles based on evolving behavioral signals.
- Detection and Model Updates: Details the process of concept drift adaptation, including the division into concept drift detection and model updates, which are key technical components for progressively updating user profiles.
- Learning Strategies: Reviews concept drift adaptation methods using discriminative learning, generative learning, and hybrid learning, providing the technical grounding for adaptive learning strategies in intelligent systems.
