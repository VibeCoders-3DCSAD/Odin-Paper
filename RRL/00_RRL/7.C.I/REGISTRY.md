## I--Alammar_et-al_2026

### Description

Frames resource allocation in Public Safety Networks (PSNs) as an 'Explanation-Regularized Resource Allocation Formulation' where explainability is a first-class design objective, transforming the learning task into a structured prediction problem.
- Hybrid Allocation Mechanism: Contrasts the proposed framework, which uses a hybrid mechanism regulated by explainability signals, with traditional approaches that involve solving a full constrained optimization problem at each step.
- Computational Tractability Evaluation: Provides a formal asymptotic complexity derivation and empirical runtime decomposition to characterize the theoretical scaling behavior and computational budget of the explainable approach, demonstrating tractability.

## I--Danach_et-al_2025

### Description

Presents a comprehensive systematic review of the intersection between explainable AI (XAI) and optimization techniques used in high-stakes decision-making.
- Interpretable Optimization Domains: Explores the systematic incorporation of interpretability into optimization-driven decision-making pipelines across various application domains like healthcare, finance, logistics, and energy systems.
- Integrates XAI and Optimization: Analyzes and classifies existing research by focusing on the integration of specific XAI methods (e.g., SHAP, LIME) with optimization strategies (e.g., genetic algorithms, reinforcement learning).

## I--Leibiker_&_Talmon_2023

### Description

Proposes an approach for participatory budgeting based on machine learning techniques, specifically a recommendation system and binary classification, to estimate voter preferences from partial ballots.
- Constrained Selection Goal: Aims to select a subset of projects that meets a total cost upper bound while considering the preferences of voters, which is characteristic of a constrained resource allocation or optimization problem.
- Evaluates Performance Metrics: Evaluates algorithms based on real-world instances regarding their accuracy in approximating voter ballots and predicting the overall outcome of the participatory budgeting process.

## I--Lotfi_et-al_2024

### Description

Frames the Budget Allocation Problem (BAP) as a hybrid robust stochastic optimization problem, incorporating risk criteria to ensure the robustness of the objective function.
- Risk Integration: Introduces risk criteria, specifically Weighted VaR (WVaR) and the minimum function, into the optimization model to handle budget fluctuations and enhance project viability.
- Sustainable and Robust Framework: Designs the model to minimize the weighted expected value, WVaR, and the minimum progress function, promoting a Sustainable, Robust, and Resilient (S3R) framework.

## I--Qu_et-al_2024

### Description

Frames existing lightweight embedding methods for recommender systems as approaches that optimize a heuristic balancing recommendation quality and memory complexity, often using an implicitly enforced memory constraint that can fail to strictly meet the requested memory budget.
- Guaranteed Budget Fulfillment: Proposes a novel method, Budgeted Embedding Table (BET), that generates table-level actions (embedding sizes for all users and items) with a guarantee of meeting pre-specified memory budgets, contrasting with prior instance-by-instance optimization.
- Efficient Action Evaluation: Employs a set-based action formulation and a DeepSets-based fitness prediction network to efficiently evaluate the recommendation quality of each table-level action, addressing the inefficiency of previous methods that require retraining the recommender system for every evaluation.

## I--Zhang_et-al_2026

### Description

Frames the budget-constrained service composition problem as a Satisfaction Optimization under Multi-user Budget Constraints (SOMBC) problem, which aims to maximize total user satisfaction under budget constraints.
- Constrained Optimization Structure: Characterizes the service composition portion of the problem as a classical combination optimization problem, often modeled as knapsack variants or a multiple-choice multiple-dimensional knapsack problem (MMKP) to select an optimal solution under budget constraints.
- Computational Challenges: Addresses the computational challenges in service composition, which includes designing an evaluation function for satisfaction, dynamically predicting user preferences (which evolve with budget conditions), and finding the global satisfaction-maximizing solution under budget constraints.
